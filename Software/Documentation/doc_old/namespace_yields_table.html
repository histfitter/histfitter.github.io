Calculate before/after-fit yields in all channels given.

Parameters:
filename The filename containing afterFit workspace
regionList A list of regions to be considered
sampleList A list of samples to be considered
dataname The name of dataset (default='obsData')
showSum Calculates sum of all regions if set to true (default=False)
doAsym Calculates asymmetric errors taken from MINOS (default=True)
blinded Observed event count will not be shown if set to True (default=False)
splitBins Calculates bin-by-bin yields for all regions if set to True (default=False)
