  lumiFB = 1.0,
  nTOYS = 3000,

Calculate before/after-fit yields in all channels given.

Parameters:
filename The filename containing afterFit workspace
poiname Name of ParameterOfInterest = POI (default='mu_SIG')
lumiFB Given lumi in fb-1 to translate upper limit on N_events into xsection limit (default='1.0')
nTOYS Number of toys to be run
asimov Boolean to run asimov or not (default=False)
wname RooWorkspace name in file (default='combined')
