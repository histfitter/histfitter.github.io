  oneSide = False,

Add a HistoSys entry using the nominal, high and low histograms, set if to include OverallSys.

If includeOverallSys then extract scale factors

If normalizeSys then normalize shapes to nominal

Parameters
systNameName of the systematic
nomNameNominal name for the systematic
highNameName of the +1sigma systematic value
lowNameName of the -1sigma systematic value
includeOverallSysInclude an overallSys for the systematic uncertainty
normalizeSysNormalize the systematic to the normRegions set through setNormRegions()
symmetrizeBoolean to indicate whether the low value has to be taken from the high value (default false)
oneSideBoolean to indicate whether the uncertainty is one-sided (default False)
samNameName of the sample
normStringString to append to the name of renormalised samples (default empty)
nomSysNameName of the nominal systematic to generate high/low from (optional use (see source); default empty)
