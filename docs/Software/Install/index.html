Introduction

The latest recommended HistFitter version can be downloaded here: HistFitter-2.0.tar.gz.

Please note that the HistFitter group can provide support only on a best-effort basis. But we are happy to hear feedback.

Installation of ROOT

Before installing HistFitter, one needs to setup ROOT.

The instructions given on the official ROOT page should be followed for this.

Be sure that when you configure the ROOT installation, you need to enable RooFit, Minuit2 and pyROOT with
--enable-roofit --enable-minuit2 --enable-python.
The recommended HistFitter version requires ROOT release 5.34.23 or greater.

HistFitter installation

After you have installed ROOT and downloaded HistFitter, for installation you need to go through the following steps:

- Untar the HistFitter package
- Setup ROOT (if not already done)
- Go the HistFitter directory cd HistFitter-2.0
- Run the HistFitter setup script source setup.sh
- Go to the src/ directory and compile the C++ side of HistFitter cd src && make

- Go back to the main HistFitter directory

You should be ready to go from here.

Every time you login into your machine and would like to use HistFitter, remember to setup ROOT and run the HistFitter setup script source setup.sh.
