Introduction

HistFitter includes a collection of tools and functions to aid the presentation of the results.
On this page, we showcase some of the plots/tables that have been produced with HistFitter.
No physics meaning should be sought in these plots/tables, they are just to show the machinery.
How to produce these, is described in detail on the Tutorial page.

Plot: Before/after fit plots

Before fit plot
This multi-bin region contains several samples, represented by different colors. The model includes several systematics, that are as the error band on top of the complete model. The difference between the data and the model is exemplified in the ratio plot.
For all regions defined in the user configuration file, these plots can be produced by a simple command-line call to the package.
.pdf
After fit plot
The same region but after running the fit of the model to the data.
.pdf

Plot: Contour plot

Exclusion contour plot
This plot is produced by running an exclusion hypothesis test on a grid of signal points belonging to the same signal model. The mass scale parameters m0 and m1/2 are varied between the signal points. The plot shows the interpolated contour for 95% CL exclusion limit and the uncertainties.
.pdf

Plot: Pull plot

Pull plot
This plot summarizes the difference between the number of observed events (nobs) and the number of predicted events (npred) for several regions. The top plot shows the number of observed/predicted events per region, while the bottom plot shows the pull , defined as the difference divided by the total uncertainty.
.pdf

Table: Event yields

Yields table
This table shows the number of observed events and the number of predicted events, before and after the fit, with the corresponding uncertainties. The number of predicted events is also shown per sample that is included in the model. Multiple options exist in the table production script to mould the output to users needs.
.pdf

Table: Systematic uncertainties

Systematics table
This table shows the breakdown of the parameters included in the model. For the systematics, nuisance parameters are introduced, while the normalization factors for the dominant backgrounds are typically treated as free floating parameters. Multiple options exist in the table production script to mould the output to users needs.
.pdf

Table: Upper Limits

Upper limits table
This table shows the model-independent upper limit on the number signal events, given the number of observed/expected events, as well as the upper limit on the visible cross section. Furthermore the discovery p-value , p(s=0), is shown. Multiple options exist in the table production script to mould the output to users needs.
.pdf
